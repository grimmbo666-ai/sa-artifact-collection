# 📊 Activity Diagram: Перемещение карточки

## Диаграмма (PlantUML)

```plantuml
@startuml
start

:Пользователь перетаскивает карточку;

if (Карточка валидна?) then (да)
  :Frontend отправляет PATCH /api/cards/{id};
  :Backend проверяет права доступа;
  
  if (Пользователь имеет права?) then (да)
    :Обновление статуса карточки;
    :Сохранение в БД;
    :Отправка WebSocket уведомления\nдругим участникам;
    :Обновление UI у всех пользователей;
  else (нет)
    :Возврат ошибки 403;
    :Отмена перемещения;
  endif
else (нет)
  :Показать ошибку валидации;
endif

stop
@enduml

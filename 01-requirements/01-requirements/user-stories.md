# 📋 User Stories

## US-01: Создание доски

**Как** пользователь,  
**хочу** создавать новую доску,  
**чтобы** организовать работу над проектом.

### Acceptance Criteria:
- [ ] Кнопка "Создать доску" на главной
- [ ] Поле для названия доски (обязательное)
- [ ] Поле для описания (опционально)
- [ ] После создания — редирект на доску

---

## US-02: Создание карточки задачи

**Как** участник доски,  
**хочу** создавать карточки задач,  
**чтобы** добавлять новые задачи в проект.

### Acceptance Criteria:
- [ ] Кнопка "Добавить карточку" в каждой колонке
- [ ] Поле "Название" (обязательное)
- [ ] Поле "Описание" (опционально)
- [ ] Поле "Дедлайн" (опционально)
- [ ] Выпадающий список "Исполнитель"

---

## US-03: Перемещение задачи

**Как** участник доски,  
**хочу** перетаскивать карточки между колонками,  
**чтобы** обновлять статус задачи.

### Acceptance Criteria:
- [ ] Drag-and-drop работает мышью
- [ ] Визуальная обратная связь при перетаскивании
- [ ] Статус задачи обновляется автоматически
